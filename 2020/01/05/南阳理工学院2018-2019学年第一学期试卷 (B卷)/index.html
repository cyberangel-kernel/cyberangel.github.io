1.关于析构函数的描述，说法正确的是（    A   ）。
A.析构函数不能重载                    B.析构函数名与类名相同
C.析构函数可以有一个或多个参数        D.析构函数的定义只能在类体内 2.基类中的成员函数，表示纯虚函数的是（    D   ）。
A.virtual void f(int){}              B.void f(int)=0;
C.virtual void f(int);               D.virtual void f()=0; 3.（    B   ）不是 C++语言的基本数据类型。
A.字符类型                            B.枚举类型
C.逻辑类型                            D.整数类型 4.要求打开文件 d:\file.dat，可写入数据，正确的语句是（    B    ）。
A.ifstream infile(“d:\file.dat”, ios::in);
B.ifstream infile(“d:\file.dat”, ios::in);
C.fstream infile(“d:\file.dat”, ios::in| ios::out)
D.ofstream infile(“d:\file.dat”, ios::out); 5.若有定义语句：float y[5]={1,2,3};，则下面描述正确的是（  A  ）。
A．y 并不代表数组的元素
B．定义此数组时不指定数组的长度，定义效果相同
C．数组含有 3 个元素
D．y[3]的值为 3 6.有如下定义语句：int x,y,z;并且 x=10,y=20,则下列程序没有输出结果的是（    C   ）。
A.cout<<”x=”<<x;                     B.cout<<”x+y=”<<x+y;
C.z=xy;                            
D.cout<<xy<<”!=”<<x-y; 7.在类定义的外部，可以被访问的成员有（  C ）。
A. 所有类成员                B. private 或 protected 的类成员
C. public 的类成员           
 D. public 或 private 的类成员 8.下面对构造函数的不正确描述是（  B  ）。
A.系统可以提供默认的构造函数      B.构造函数可以有参数，所以可以有返回值
C.构造函数可以重载                  D.构造函数可以设置默认参数
9.C++可以采用动态内存分配在运行时为指针分配内存。下面（   C   ）对一个 100 个元素的动态数组进行动态内存分配和释放。
A.int *pi = new int(100); delete ()pi; 
B.int *pi = new int; delete [] pi;
C.int *pi = new int[100]; delete[] pi;
D.int *pi = new int; delete pi;
10.int fun(int,int);不可与下列哪个函数构成重载（   D    ）。
A.double fun(int,double);            B.double fun(double,double);
C.int fun(int,int,int);              D.double fun(int,int);

1.当建立对象时，程序自动调用该类的         构造函数      ，当对象生命周期结束时自动调用该类的析构函数。 2.设有语句 int x[8]={8,7,6,5,2,1},*s;s=x+3;则执行 cout<<*s 语句的结果为       5      。 3. 如果要声明一个普通函数为类的友元函数，需要在该函数前使用关键字       friend      。 4. 设有语句”char s[]=”china\0nanjin”;int m=strlen(s);”则 m 的值为       5       。 5. 执行完下列三条语句后，指针变量 c 指向       b      。
int a,b,_c=&a;
int _&p=c;
p=&b; 6.在函数原型声明中，必须声明函数参数的类型，但可以省略       参数名        。
7.  如果一个函数直接或间接地调用自身，这样的调用称为       递归        调用。 8.声明静态数据成员时前边要使用关键字       static      。 9.对一个二维字符数组 a 进行初始化的数据为{“123”,“456”,“789”}，则 a[1]元素对应的字符串为       456      。 10.一个完全二叉树总共有 25 个结点，那该二叉树的深度是      5       。

1.C 语言程序中可以有多个函数，但只能有一个主函数。（     √    ） 2.设有定义一维数组如下：int a[5],*p=a;则表达式 p=p+1 是合法的。（     √    ） 3.构造函数可以声明为虚函数。（     ×   ）
4.case 语句后如没有 break，顺序向下执行。（     √    ） 5.在说明语句”int a(15),&b=a,*p=&a;”中，b 的值与*p 的值是相等的。（     √    ） 6.用 new 运算符动态分配的存储空间若不用 delete 释放， 则程序运行结束后该空间会自动释放。（     ×    ） 7.一个类的友元关系可以被继承。（     ×    ） 8.析构函数能被声明为虚函数。（     √    ）
构造函数不可声明为虚函数，而析构函数可以声明为虚函数。 9.派生类的成员函数不能直接访问基类的私有成员。（     √    ） 10.类型向上兼容原则是指派生类指针可以指向基类对象。（     ×    ）
