C++11 auto和decltype关键字

Created2020-01-15|Updated2021-07-04
|Post View:

可以用 auto 关键字定义变量，编译器会自动判断变量的类型。例如：

  1. auto i =100; // i 是 int
  2. auto p = new A(); // p 是 A*
  3. auto k = 34343LL; // k 是 long long

有时，变量的类型名特别长，使用 auto 就会很方便。例如：

  1. map <string, int, greater >mp;
  2. for( auto i = mp.begin(); i != mp.end(); ++i)
  3. cout << i -> first << “, “ << i -> second;

编译器会自动判断出 i 的类型是 map<string, int, greater >::iterator。

decltype 关键字可以用于求表达式的类型。例如：

  1. int i;
  2. double t;
  3. struct A { double x; };
  4. const A* a = new A();
  5. decltype(a) x1; //x1 是 A*
  6. decltype(i) x2; //x2 是 int
  7. decltype(a -> x) x3; // x3 是 double

在上面的例子中，编译器自动将 decltype (a) 等价为A*，因为编译器知道 a 的类型是A*。

auto 和 decltype 可以一起使用。例如：

  1. #include
  2. using namespace std;
  3. struct A {
  4. int i;
  5. A(int ii) : i(ii) {}
  6. };
  7. A operator + (int n, const A & a)
  8. {
  9. return A(a.i + n);
  10. }
  11. template <**class** T1, **class** T2>
  12. auto add(T1 x, T2 y) -> decltype(x + y) {
  13. return x + y;
  14. }
  15. int main() {
  16. auto d = add(100, 1.5); // d 是 double 类型，d = 101.5
  17. auto k = add(100, A(1)); // k 是 A 类型，因为表达式“100+A(1)”是 A 类型的
  18. cout << d << endl;
  19. cout << k.i << endl;
  20. return 0;
  21. }

程序的输出结果如下：
101.5
101

第 12 行告诉编译器，add 的返回值类型是decltype(x+y)，即返回值的类型和x+y这个表达式的类型一致。编译器将 add 实例化时，会自动推断出x+y的类型。

在  C++11 中，函数返回值若为 auto，则需要和 decltype 配合使用。在 C++14 中，则可以不用 decltype，例如下面的程序没有问题：
纯文本复制

  1. auto add (int a, int b)
  2. {
  3. int i = a + b;
  4. return i;
  5. }
