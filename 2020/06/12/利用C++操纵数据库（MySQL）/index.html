二、修改配置

将文件解压到 C 盘的根目录下：

双击打开文件夹：

在目录下创建一个新文件“my-default.ini”（注意这里的扩展名），打开编辑并保存：

[mysqld]

设置 3306 端口

port=3306

设置 mysql 的安装目录，一定要与上面的安装路径保持一致

basedir=C:\mysql-8.0.20-winx64

设置 mysql 数据库的数据的存放目录，自动生成，无需手动创建，当然也可以放在其他地方

datadir=C:\mysql-8.0.20-winx64\Data

允许最大连接数

max_connections=200

允许连接失败的次数。

max_connect_errors=10

服务端使用的字符集默认为 utf8mb4

character-set-server=utf8mb4

创建新表时将使用的默认存储引擎

default-storage-engine=INNODB

默认使用“mysql_native_password”插件认证

#mysql_native_password
default_authentication_plugin=mysql_native_password
[mysql]

设置 mysql 客户端默认字符集

default-character-set=utf8mb4
[client]

设置 mysql 客户端连接服务端时默认使用的端口，不建议修改，这是公认端口号

port=3306
default-character-set=utf8mb4

默认 utf8 格式可以不写，但是为了避免以后由于编码出现的问题可以在这先设置

data 文件夹是未被创建的文件夹，不需要自己手动创建

四、安装 MySQL 服务

请以管理员身份运行“命令提示符”，进入到 MySQL 的 bin 目录下，执行命令：

cd C:\mysql-8.0.20-winx64\bin

初始化执行：

mysqld –initialize

若报错“缺少 vcruntime140_1.dll”，请将附件下载并放入到 bin 目录下
此时你的 MySQL 根目录下会生成一个 data 文件夹，如果没有生成 data 文件夹，则输入下面代码

mysqld –initialize-insecure –user=mysql

安装服务

mysqld –install

启动服务

net start mysql


可以在计算机管理中找到此项服务：

六、修改密码

cmd 执行：mysql -u root -p
使用刚才的初始密码进行登录:

修改密码:

ALTER USER ‘root‘@’localhost’ IDENTIFIED BY ‘新密码’;

比如将密码修改为 root:

ALTER USER ‘root‘@’localhost’ IDENTIFIED BY ‘root’;


修改成功!
可以使用 navicat 的图形工具对进行数据库管理:

七、Navicat 的图形工具

将附件下载下来,打开安装包进行安装

一直点击下一步就行了…
打开破解补丁,找到安装目录,我的是:

C:\Program Files (x86)\PremiumSoft\Navicat for MySQL

双击 navicat.exe,破解成功!

打开软件:
点击 MySQL

点击确定,双击红色箭头处

若弹出窗口:

管理员:cmd 执行：mysql -u root -p
输入密码登录数据库之后:

输入
ALTER USER ‘root‘@’localhost’ IDENTIFIED WITH mysql_native_password BY ‘root’;
然后按 Enter 键，其中 root 是用户名，localhost 特指本地主机，mysql_native_password 是旧的密码校验机制，root 是密码

再输入 FLUSH PRIVILEGES;然后按 Enter 键,此时我们再用 Navicat Premium 连接、访问该数据库时，就正常了


以上就是安装步骤及解决错误的办法,若出现更多的问题,请百度…
