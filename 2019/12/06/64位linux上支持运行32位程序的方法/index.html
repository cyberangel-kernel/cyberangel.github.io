64位linux上支持运行32位程序的方法

Created2019-12-06|Updated2021-07-04
|Post View:

在 64 位的 Linux 上运行 32 位程序的时候会出现这种情况：
（1）执行 bin 文件时提示：No such file or directory
（2）ldd bin 文件   的输出为： not a dynamic executable
（3）file bin 文件 的输出显示程序是 32 位

解决方法：
debian 上只要安装 ia32-libs 这个包（apt-get install ia32-libs)就可以了。
$ sudo apt-get install ia32-libs
若无法安装，找不到库，就用下面这个方法：打开多架构支持，然后更新，在终端直接执行命令
$ sudo dpkg –add-architecture i386
$ sudo apt-get update
$ sudo apt-get install ia32-libs
如果提示没有 ia32-libs，那就用执行：
$ sudo dpkg –add-architecture i386
$ sudo apt-get update
$ sudo apt-get upgrade
$ sudo apt-get install lib32ncurses5 lib32z1

执行某一个文件就切换到那个文件所在的目录，然后执行：./文件名
