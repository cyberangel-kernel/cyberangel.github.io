DS寄存器和CS寄存器有什么区别？

Created2019-11-24|Updated2021-07-04
|Post View:

CS 是 CPU 执行当前指令的段地址，DS 是数据开始的段地址。
通俗的来讲，CS 是告诉 CPU，去哪个位置找内容当成指令去执行
DS 是告诉 CPU，去哪个位置找内容当成数据被使用
CPU 要执行 CS 中的指令 指令用到的数据就可能存放在 DS 中
你可以把数据放到 CS 中，但是 CPU 并不把它当成数据来使用，
你也可以把指令放到 DS 中，到时 CPU 根本不去 DS 里读指令

总之：DS 与数据有关，CS 与指令有关
